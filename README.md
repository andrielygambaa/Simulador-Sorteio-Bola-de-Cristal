# 🔮 Simulador de Sorteio — Bola de Cristal

✨ Um projeto web interativo desenvolvido com **HTML, CSS e JavaScript** que simula um sorteio inteligente usando uma **bola de cristal virtual**.

Neste projeto, o usuário escolhe um **número da sorte entre 1 e 10**, e o sistema utiliza um algoritmo que continua sorteando números aleatórios até acertar o número escolhido. No final, o programa revela **quantas tentativas foram necessárias para descobrir o número**.

---

# 🌌 Demonstração do Projeto

A aplicação funciona como uma **bola de cristal digital** que tenta adivinhar o número escolhido pelo usuário.

💡 O sistema realiza sorteios automaticamente até acertar o número correto.

Quando isso acontece, a bola de cristal revela:

🔢 Quantas tentativas foram necessárias
🔮 O resultado da "previsão"
✨ Uma experiência visual divertida e interativa

---

# 🧠 Conceitos Praticados

Este projeto ajuda a desenvolver diversos conceitos importantes de programação:

🔁 **Laços de repetição (while)**
O sistema continua tentando até acertar o número escolhido.

🎲 **Geração de números aleatórios**
Uso da função `Math.random()` para simular sorteios.

🧩 **Lógica de programação**
Comparação entre o número sorteado e o número escolhido pelo usuário.

⚡ **Manipulação do DOM**
Atualização do resultado diretamente na página.

🎨 **Estilização com CSS**
Interface temática inspirada em uma **bola de cristal mágica**.

# 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais da web:

💻 **HTML5**
Estrutura da página e elementos da interface.

🎨 **CSS3**

---

# 🎮 Como Usar

1️⃣ Digite um **número da sorte entre 1 e 10**
2️⃣ Clique no botão **Tentar a Sorte**
3️⃣ A bola de cristal começará a "prever" o número
4️⃣ O sistema continuará sorteando até acertar
5️⃣ O resultado mostrará **quantas tentativas foram necessárias**

🔮 Quanto maior a sorte… menos tentativas!
